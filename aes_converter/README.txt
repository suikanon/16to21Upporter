USAGE:
Place PES16 aes exports in exports_to_convert21
Place PES16 EDIT savefile used with aes exports to convert in exports_to_convert21
Set correct team ids in the teams_list_XX.txt files if required.
Double click convert_exports.bat to execute it
Converted exports will be generated in the exports_converted21 folder

KNOWN ISSUES:
.model files only found in a PES16 Boots folder are not supported yet
Materials using texture samplers other than DiffuseMap are not supported yet
Facial skeletons are not supported yet
